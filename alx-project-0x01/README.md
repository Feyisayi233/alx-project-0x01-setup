# alx-project-0x01

This project was created using Next.js with TypeScript, ESLint, Tailwind CSS, and import alias. The /src directory and App Router are not used.

## Getting Started

To start the development server on port 3000:

```
npm run dev -- -p 3000
```

## Features

- TypeScript
- ESLint
- Tailwind CSS
- Import alias (@/\*)

## Project Structure

- No /src directory
- No App Router
